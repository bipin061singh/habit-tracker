# Habit Tracker

A minimal habit tracking application built with React.

## Features
- Add and track daily habits
- Mark habits as complete
- View habit streaks
- Simple, clean UI

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

Open http://localhost:3000 to view it in the browser.

### Running Tests

```bash
npm test
```

### Building for Production

```bash
npm run build
```

## Tech Stack
- React 18
- Create React Scripts
- React Testing Library
